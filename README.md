
## Examen

El proyecto contiene una SPA hecho con Vuejs y Laravel.

# Caracteristicas
- Autenticacion y registro con JWT.
- Listado de productos/comidas.
- Agregar y eliminar productos del carrito.
- Generar Pedido.
- Listar Pedidos.
- Ver detalle del pedido.
